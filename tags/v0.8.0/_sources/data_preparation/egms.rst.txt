**EGMS**
^^^^^^^^

  `EGMS <https://egms.land.copernicus.eu/>`_ allows downloading the data in csv format. You can add the data to QGIS as a `delimited text layer <https://docs.qgis.org/3.34/en/docs/user_manual/managing_data_source/supported_data.html#delimited-text-files>`_.

  The imported layer has the following fields:

  .. list-table::
    :header-rows: 1

    * - Field Name
      - Description
    * - ``mean_velocity``
      - A field containing the InSAR velocity data.
    * - ``YYYYMMDD``
      - Multiple fields for time series data.
