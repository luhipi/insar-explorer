about=InSAR Explorer is a QGIS plugin designed for interactive visualization and analysis of InSAR time series results. With a user-friendly interface, it allows users to dynamically plot and explore ground displacement data over time. You can find the source code on GitHub (https://github.com/luhipi/insar-explorer). A sample shapefile containing time series data for testing the plugin is available on Zenodo (https://zenodo.org/records/14052814). Currently, vector data from SARvey, MintPy, MintPy, and EGMS and raster data from GMTSAR are supported. Please refer to the documentation for more information (https://luhipi.github.io/insar-explorer/)
